## 2. 훈련 철학 — 핵심 원칙

### 2.1 Bakken 노르웨이 모델의 마라톤 적용

- **원래 모델**: 80% Z1 이지 + 20% Z2 역치, VO2max(Z3)는 베이스기에 거의 없음
- **학술적 분류**: 순수 Polarized가 아닌 Pyramidal에 가까움 (Z1 > Z2 > Z3 볼륨 분포)
- **마라톤 적용 변형**: Bakken 본인 언급 — "1500m~10km는 순수 인터벌이 우월하지만 마라톤에는 확실히 더 많은 continuous work 필요"
- **핵심 원리**: 버틸 수 있는 한도 내에서 역치를 가져간다. 4.0 mmol/L이 아닌 **2.3~3.0 mmol/L**에서 대량 볼륨 축적
- **한 문장 요약**: "역치를 넘지 않으면서, 역치에 가장 가까운 곳에서, 가장 오래 버티는 것"

### 2.2 경사도 트레드밀 전략

- 경사도 8%, 심박 기반 강도 제어
- **목적**: 같은 심박(심혈관 부하)을 더 느린 속도(낮은 충격)로 달성
- 평지에서 Z4(150~160)를 만들려면 4:50~5:00/km 필요 (착지 충격 체중의 2.5~3배)
- 경사도 8 트레드밀에서는 6:30/km로 같은 심박 달성 (충격 대폭 감소)
- **원칙**: "심혈관 부하는 높게, 충격은 낮게" — 40세 부상 이력 러너의 장기 지속 가능성 핵심

### 2.3 "참는 근육" 훈련

- 즉흥적·감각적 충동 제어가 시스템의 핵심 근육
- "더 갈 수 있다"는 감각이 가장 위험 (젖산 이미 상승, 체감은 지연)
- 좋은 날에 더 하는 게 아니라, 좋은 날의 여유분을 다음 세션의 품질로 가져감

### 2.4 페이스 vs 심박 기준

- 바람·기온·피로 등 외부 조건 변화 시 페이스는 자연스럽게 조정
- **심박 기준 충족이 세션 성공 여부 판단 기준**
- "오늘 느렸네" → "조건 고려 시 적정 강도였다" 사고 전환

### 존 설정 배경

- **초기 가정**: LT1 = 128 (HRmax 70%), LT2 = 158~163
- **문제 발견**: 마라톤 3:29를 평균 HR 160으로 완주 → LT1이 128이면 3.5시간 역치 이상은 생리학적으로 불가능
- **재조정**: LT1을 150으로 상향 → 마라톤 시 대부분 LT1~LT2 사이 = 생리학적으로 자연스러움
- **정밀 측정 필요**: GXT(점증부하운동검사)로 LT1/LT2 심박 확인 권장 (6~8주마다)

### 4.4 핵심 발견: "찍는 것과 머무는 것의 차이"

- **전환 전**: Z4를 순간적으로 찍고 빠지는 톱니 형태 → 실제 역치 자극 시간 짧음
- **전환 후**: Z4 하단(144~150)에서 수평 유지 → 분 단위 지속적 적응 자극
- **전환 방법**: 속도를 올려 심박을 Z4 하단까지 끌어올린 후, 속도를 내리고 심박 관성으로 Z4 안에 머무르기 ("파도타기")
- **결과**: 같은 세트 수에서 Z4 비율 50% → 71~74%로 증가

## 6. 세션 종료 기준

역치 세션은 체감이 아닌 **사전 규칙**으로 끊어야 함 ("아프지 않으니까 계속할 수 있다"가 가장 위험한 함정):

1. **사전 설정 세트 수**: 시작 전에 정한 세트 수에서 끊음. 좋은 날에도 추가하지 않음
2. **심박 천장 규칙**: 설정한 천장(161)에 도달하는 세트 → 해당 세트를 마지막으로 종료
3. **Cardiac drift 규칙**: 초반 대비 같은 속도에서 심박 +5bpm 이상 → 종료 신호
4. **Rest 심박 규칙**: 1분 rest에서 심박이 125 아래로 안 떨어지기 시작하면 → 종료 신호

### 싱글 세션 적정 볼륨

- 평일 싱글: 5~6세트 (Z4 약 20분)
- 주말 싱글: 7~8세트 (Z4 약 30~35분)
- **볼륨 증가 = 세트 수/세션 수 증가, 단일 세션 Z4 시간 무한 연장이 아님**

## 10. 내리막 충격 관리

- 오르막은 충격 적음, **내리막은 평지보다 충격이 더 큼**
- 대퇴사두근 편심 수축(eccentric) 부하 집중
- **대응법**: 케이던스를 높여 보폭 단축 → 착지점이 무게중심 가까이 → 브레이크 충격 최소화
- 실제 트레일러닝 데이터에서 최대 케이던스 203spm 확인 (내리막 구간)

## 15. 핵심 인사이트 요약
